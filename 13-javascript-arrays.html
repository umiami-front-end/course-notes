JavaScript Arrays

Arrays

  • stores an ordered list of values
  • values may be any data type (even other arrays or objects)
["cat", 25, true]

Array Literal

  • values are placed within brackets ([])
  • values may be inline or on individual lines
  • separated by commas
  • last value does not require a comma
var lieDetector = [
  true,
  false,
  false
];

Array Constructor

Values are provided as arguments to Array(). Values may also be placed on individual lines.

var petStore = new Array("cats", "dogs", "fish");

Array Index

Arrays are indexed. The index begins at 0 (zero).

["cat", "dog", "fish"]
item index value
cat 0
dog 1
fish 2

Array Length

Since array items are indexed, we can also determine the size of an array.

var petStore = ["cat", "dog", "fish"];
petStore.length;  // 3

Accessing Array Items

We can access individual items in the array using bracket notation.

var petStore = ["cat", "dog", "fish"];
var num = 0;

petStore[0];    // cat
petStore[1];    // dog
petStore[2];    // fish

petStore[num];  // cat

The Last Item

JavaScript does not provide an elegant way to return the last item of an array. We can create our own solution.
