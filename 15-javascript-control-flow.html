Control Flow

Comparing Expressions

Imagine we are a police officer using a script to figure out which drivers need to be pulled over.

var driverSpeed = 50;
var speedLimit = 65;
driverSpeed >= speedLimit  // false

Combining Expressions

When testing two or more expressions, enclose them in parentheses and use a logic operator to return a final true or false value.

var driverSpeed = 85;
var speedLimit = 65;
var workingTailLight = false;

(driverSpeed >= speedLimit) ||
  (workingTailLight == false);  // true, true

if statements

An if statement checks a conditition.

If the return value is true, the statement(s) within the code block are executed. If the return value is false, nothing happens.

var driverSpeed = 85;
var speedLimit = 65;

if (driverSpeed >= speedLimit) {
  console.log("Issue a ticket.");
}

if-else statements

What if we want something to happen when the return value is false?

An if-else statement, true will execute the statements in the first code block; false will execute statements in the last code block.

if (driverSpeed >= speedLimit) {
  console.log("Pull over, driver!");  // true
} else {
  console.log("Eat another donut.");  // false
}

Multiple if-else statements

We can also test a situation using multiple if-else statements.

if (driverSpeed >= speedLimit) && (!workingTailLight) {
  console.log("Issue a ticket.");
} else if (workingTailLight == false) {
  console.log("Issue a warning.");
} else {
  console.log("Eat another donut.");
}

switch-case statements

Multiple if-else statements can be coded as a switch-case statement.

var color = prompt("Your favorite color is...");

switch (color) {
  case 'red':
    console.log("Red is fire.");
    break;

  case 'yellow':
    console.log("Yellow is sunshine.");
    break;

  case 'green':
    console.log("Green is nature.");
    break;

  case 'blue':
    console.log("Blue is water and sky.");
    break;

  default:
    console.log("Not feeling colorful today?");
    break;
}

switch-case anatomy

The switch-case statement can be complex. Let's break down each part.

switch

switch (condition) { }

Begin the statement with the switch keyword followed by a condition in parentheses. The condition may be an expression or just a variable.

Curly brackets create a code block which surrounds the cases.

case

case 'red':
  console.log("Red is fire.");
  break;

There are three parts to specifying each case:

  1. case keyword, a value to match and a colon (:).
  2. statements to be executed
  3. break statement to exit the switch-case when a match is found

default

default:
  console.log("Not feeling colorful today?");
  break;

The default is the last 'case' in the code block which is executed if no match is found. A break is still required to exit the switch-case statement.

switch-case benefits

A switch-case statement is best used when:

  • multiple conditions exist (4-5)
  • user input may vary
  • a default value is necessary
  • speed is essential
