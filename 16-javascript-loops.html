Loops

What is a loop?

A loop is a code block which repeats as long as a true condition is met.

Infinite Loops

An infinite loop is a loop which repeats without end. It holds up the program from continuing.

Always make sure your loops return a false value at some point.

for loop

A for loop is executes code a specific number of times.

for (i = 0; i < 3; i++) {
  console.log("Hello, world!");
}

The loop above will print "Hello, world!" to the console three times.

for loop counter

Within the parentheses next to the for keyword is a counter which tracks each time the loop runs.

(i = 0; i < 3; i++)
  • i = 0 initializes our counter to start at 0.
  • i < 3 sets the condition to return true/false
    • when i is less than 3, the loop continues
    • when i is greater than or equal to 3, the loop stops
  • i++ increments i after the statement inside the loop is executed

while loop

A while loop is used when the number times a loop should execute is unknown. These are susceptible to becoming infinite loops; it is easy to forget incrementing the counter.

i = 1;

while (i < 3) {
  console.log("Greetings!");
  i++;  // do not forget me!
}

The loop above will print "Greetings!" to the console twice.

do-while loop

A do-while loop is used when a loop should execute at least once, even if a true condition is never met.

var i = 1;

do {
  console.log("Hi!");
  i++;
} while (i < 1);

The loop above will print "Hi!" to the console once.

Note how the do-while loop tests to condition after the code block is executed.

for vs. while

If the amount of data to be received from an external source is unknown, use a while loop.

Examples of when to use a while loop instead of a for loop:

  • user input is optional
  • an API

break & continue

When control flow (if, if-else, switch-case) are included within a loop, it may be necessary to stop the loop â€” even if the counting condition we set initially is not met.

The break keyword forces the loop to quit.

The continue keyword tells the loop to carry on.

Loops & Arrays

A common use of a loop is to process data in an array. This is called iteration.

Given an array of names, we might want to include each in a greeting.

var names = ["Flynn", "Quorra", "Charlie"];

This is the hard way of doing it:

console.log("Hello, " + names[0]);
console.log("Hello, " + names[1]);
console.log("Hello, " + names[2]);

Iterating Over an Array

How about this instead?

var names = ["Flynn", "Quorra", "Charlie"];
var i = 0;
while (i < names.length) {
  console.log("Hello, " + names[i]);
  i++;
}

Iteration Steps

  1. Declare the array
  2. Set a counter
    • Start at 0 since our array index begins at 0
  3. Set the condition
    • "While the counter is less than the number of items in the array..."
  4. Write a message which includes each name
    • Notice i is used to indicate which index of the array to call!
  5. Increment the counter

Iterating Over an Object

It is also possible to iterate over an object.

The loop will execute once for each property of the object.

var car = {
  year: 2016,
  make: "Chevrolet",
  model: "Camaro"
}

for (var property in car) {
  document.write(car[property] + " ");
}
